# 📖 Petrarca Workspace
Petrarca Workspace is a student suite to make studying easier for the whole world. The available languages ​​are limited Italian & English but we will add others in the future. Your data is safe thanks to our database! All your data will not be lost and will remain encrypted and safe!
The account will be synchronized with all our services!

## ✨ Main features
The workspace includes:

📅 A diary to note down all the school statistics

📝 A note manager to keep track of all the main notes

📃 A document editor to create amazing docs

🖼️ A slide editor to create amazing presentations

And much more to come!!
